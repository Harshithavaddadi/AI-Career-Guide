# 🚀 AI Career Guide

AI Career Guide is a full-stack AI-powered web application that helps students and job seekers explore career paths, analyze skills, and get personalized job recommendations using AI-based insights.

---

## 📌 Features

- 🎯 AI-based career recommendations
- 📊 Domain & skill analysis
- 💼 Job suggestions based on profile/resume
- 📄 Resume input & evaluation
- ⚡ Fast and responsive UI
- 🌐 Full-stack architecture (Frontend + Backend)

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Vite
- CSS / Tailwind CSS

### Backend:
- Node.js
- Express.js

### AI Integration:
- Gemini AI / AI APIs (for recommendations)

### Other Tools:
- REST APIs
- Git & GitHub
